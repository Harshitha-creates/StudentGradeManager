# StudentGradeManager

A simple Java console-based application to store student names, take marks for 5 subjects, calculate averages, assign grades, and display all student details.  
This project is built using **Core Java**, **ArrayList**, **functions**, and **basic Object-Oriented Programming (OOP)** concepts.
